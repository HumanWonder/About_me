Présentation

Ce projet est une recréation simple de Space Invaders réalisée en JavaScript natif. Aucun canvas ou bibliothèque externe n’a été utilisé. Le rendu repose uniquement sur la manipulation du DOM (éléments HTML et styles CSS).

Fonctionnalités principales

  • Déplacement du joueur avec les touches fléchées.
  • Tirs et gestion des projectiles.
  • Animation et mouvement des ennemis.
  • Système simple de score et détection des collisions.

Exemple de code

const player = document.createElement('div');
player.classList.add('player');
document.body.appendChild(player);

document.addEventListener('keydown', (e) => {
    if (e.key === 'ArrowLeft') moveLeft();
    if (e.key === 'ArrowRight') moveRight();
});
